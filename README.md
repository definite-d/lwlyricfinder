# lwlyricfinder

**lwlyricfinder** is a simple command-line tool that scrapes [LoveWorld Lyrics](https://loveworldlyrics.com/) for song lyrics and copies them to your clipboard.

## Features
- Fetch lyrics for a given song.
- Automatically copies lyrics to the clipboard.
- Lightweight and easy to use.

## Installation
```sh
# Clone the repository
git clone https://github.com/yourusername/lwlyricfinder.git
cd lwlyricfinder

# Install dependencies
pip install -r requirements.txt
```

## Usage
```sh
python lwlyricfinder.py "Song Name"
```
